# ======================================
# INSTRUÇÕES PARA CRIAR KEYSTORE
# ======================================

IMPORTANTE: Execute estes comandos NO TERMINAL (não aqui no Claude)

## PASSO 1: Criar o Keystore

Execute no terminal (PowerShell ou CMD):

cd C:\Users\alext\Downloads\nutritracker\nutritracker\android

keytool -genkey -v -keystore upload-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias upload

## O comando vai perguntar:

1. "Enter keystore password:" 
   → Digite uma SENHA FORTE (ex: NutriTracker2025#Secure)
   → ANOTE ESTA SENHA!

2. "Re-enter new password:"
   → Digite a mesma senha novamente

3. "What is your first and last name?"
   → Digite seu nome (ex: Alex Tracker)

4. "What is the name of your organizational unit?"
   → Digite: NutriTracker Development

5. "What is the name of your organization?"
   → Digite: NutriTracker

6. "What is the name of your City or Locality?"
   → Digite sua cidade

7. "What is the name of your State or Province?"
   → Digite seu estado

8. "What is the two-letter country code for this unit?"
   → Digite: BR

9. "Is CN=..., correct?"
   → Digite: yes

10. "Enter key password for <upload>"
    → Aperte ENTER (usa a mesma senha)

## PASSO 2: Criar key.properties

Execute no terminal:

cd C:\Users\alext\Downloads\nutritracker\nutritracker\android

echo storePassword=SUA_SENHA_AQUI > key.properties
echo keyPassword=SUA_SENHA_AQUI >> key.properties
echo keyAlias=upload >> key.properties
echo storeFile=upload-keystore.jks >> key.properties

SUBSTITUA "SUA_SENHA_AQUI" pela senha que você criou!

## PASSO 3: Fazer backup do keystore

MUITO IMPORTANTE: Copie o arquivo "upload-keystore.jks" para um local seguro!
Se você perder este arquivo, NUNCA mais poderá atualizar o app na Play Store!

Sugestões de backup:
- Google Drive (criptografado)
- Pendrive em local seguro
- Gerenciador de senhas (1Password, LastPass)
